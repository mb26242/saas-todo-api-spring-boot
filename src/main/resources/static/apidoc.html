Welcome to the Task Management API

Here are the available endpoints:

Task Management

/users/{userId}/tasks

Manage all tasks associated with a user.

  • GET /users/{userId}/tasks - Get all tasks for a specific user.

    • Returns a list of tasks assigned to the user specified by {userId}.

  • POST /users/{userId}/tasks - Create a new task for a user.

    • Creates a new task for the user identified by {userId}.

/users/{userId}/tasks/{taskId}

Manage a specific task assigned to a user.

  • GET /users/{userId}/tasks/{taskId} - Get details of a specific task for a user.

    • Returns the details of the task identified by {taskId} for the user identified by {userId}.

  • PUT /users/{userId}/tasks/{taskId} - Update a specific task for a user.

    • Updates the task identified by {taskId} for the user identified by {userId}.

  • DELETE /users/{userId}/tasks/{taskId} - Delete a specific task for a user.

    • Deletes the task identified by {taskId} for the user identified by {userId}.

Other Resources

  • GET /users - View all users
  • GET /companies - View all companies

Utilities

  • GET /populate - Populate the repositories with initial data
  • GET /emptyRepositories - Delete all the data from the repositories
